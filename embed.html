Votecraft Widget - Embed Example

This is how the widget will look when embedded on your WordPress site.

Embed Code for WordPress

Copy this code into an HTML block in Elementor:

<iframe
    src="https://your-hosted-url.com/index.html"
    width="100%"
    height="650"
    frameborder="0"
    style="border-radius: 16px; max-width: 480px; display: block; margin: 0 auto;"
></iframe>
